# Contributing to AVS

Thank you for your interest in improving the AI Visibility Score standard.

## How to Contribute

### Propose New Checks

1. Open an issue describing the check
2. Explain what it measures and why it matters for AI visibility
3. Provide evidence (research papers, empirical data, or logical argument)
4. If possible, include a regex/HTML-parsing implementation

### Report Issues with Scoring

If you find a website that scores unexpectedly high or low:
1. Share the URL and the AVS score
2. Explain why you think the score is incorrect
3. Suggest which checks might need adjustment

### Run Validation Studies

Independent validation is the most valuable contribution:
1. Choose an AI search engine (Perplexity, ChatGPT, Gemini, Copilot)
2. Submit queries and collect cited URLs
3. Scan cited URLs with the AVS reference implementation
4. Share your findings (raw data + analysis)

### Translate the Spec

The spec is currently in English. Translations to any language are welcome.

## Code of Conduct

Be respectful. Be constructive. Focus on data, not opinions.
